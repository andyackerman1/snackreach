Method 3: Run Local Server (Full Functionality)

For full functionality locally, you need to run the server:

  1. Install Node.js: https://nodejs.org/
  2. Open Terminal and run: 
    cd /Users/andy.ackerman/snackconnect/backend
npm install
node server.js
  3. Then open: http://localhost:3000

Why Do I Need Node.js?

This application has:

  • Frontend: HTML/CSS/JavaScript (can view directly)
  • Backend: Node.js server that handles:
    • User registration and login
    • Database storage
    • API endpoints
    • Authentication

The backend server is required for login, signup, and data storage to work.
