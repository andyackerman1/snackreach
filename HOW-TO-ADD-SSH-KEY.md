# How to Add SSH Key to GitHub

## Step-by-Step Instructions

### Step 1: Open GitHub SSH Keys Page
- Go to: **https://github.com/settings/keys**
- (I've opened it for you)

### Step 2: Click "New SSH key" Button
- You'll see a green button that says **"New SSH key"** or **"Add SSH key"**
- Click it

### Step 3: Fill in the Form
- **Title:** Type `SnackReach Mac` (or any name you want)
- **Key type:** Select **"Authentication Key"** (should be default)
- **Key:** Paste the SSH key (it's in your clipboard!)

### Step 4: Paste the Key
The key is already copied to your clipboard. Just:
- Click in the "Key" text box
- Press **Cmd+V** (or right-click â†’ Paste)
- You should see a long string starting with `ssh-ed25519`

### Step 5: Click "Add SSH key"
- Scroll down and click the green **"Add SSH key"** button
- You might be asked to enter your GitHub password to confirm

### Step 6: Done!
- You'll see a success message
- The key will appear in your list
